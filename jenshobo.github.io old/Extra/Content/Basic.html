Commodore 64 game.
A game I made for the Commodore 64, it is completely written in the BASIC programming language. This is a project I started in between school projects. It was made in about 2 days which includes learning the language again.

About the game itself:
In the game your spaceship moves down, on your way are obstacles that you will need to dodge. You can if you wish destroy these obstacles, doing so will grand you points, but the game will progress even if you don't. If you destroy these obstacles they will explode, so these you still need to avoid. As you go down you will pick up speed until a certain point where you will be shown your current score. After pressing the fire button you go on to level 2 where you will find more obstacles and the game repeats. This goes on until you hit one of the obstacles.

The game was made quite a while ago now, which is a shame as I lost the program file now. Due to this the only things I have from it are the video and a single photo.
