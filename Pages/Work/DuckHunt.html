DuckHunt

During my internship at Brin I started a number of side projects, most of them I dropped later on as these where too big to create by a single person. However, one project I started I did finish, that being Duck hunt (1987) remade for the Oculus Quest in Unity (C#). For the game I used assets made by other people to get the used sprites, models, UI, sound and music. All the code which makes the game playable of course was written by me. Without the code there would be no game, then it would just be a weird collection of random sounds and pictures.

What made this project quite fun was picking apart the old game on NES, as I own this game in my collection this made it easy to pickup and figuring out how the game works. I would then go on to recreate what I found in C# and test it on my VR headset. To see what does and what doesn't work. After the main gameplay elements from the original where implemented I was able to make it look good with trees, grass and a good UI.
